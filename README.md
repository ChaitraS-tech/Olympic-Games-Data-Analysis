#  Olympic Games Data Analysis (Python)

# Project Overview

This project focuses on analyzing historical Olympic Games data using Python to uncover meaningful insights. The analysis combines Summer and Winter Olympics datasets along with country-level statistics to study medal distribution, participation trends, and performance patterns.

---

# Objectives

* Analyze historical Olympic data efficiently
* Identify trends in medal distribution and participation
* Compare performance across countries and athletes
* Study gender participation over time
* Perform exploratory data analysis using Python

---

# Tools & Technologies Used

* Python
* Pandas (Data Analysis)
* NumPy (Numerical Computing)
* Matplotlib & Seaborn (Data Visualization)
* Jupyter Notebook

---

# Dataset Description

The dataset includes the following files:

* Summer Olympics dataset
* Winter Olympics dataset
* Country-level statistics dataset

Key fields:

* Year
* Country
* Sport
* Athlete Name
* Gender
* Event
* Medal (Gold, Silver, Bronze)
* Participation Details

---

# Data Processing

1. Imported datasets into Jupyter Notebook
2. Merged Summer and Winter Olympics datasets
3. Integrated country-level data
4. Performed data cleaning:

   * Handling missing values
   * Removing duplicates
   * Standardizing data formats
5. Conducted exploratory data analysis (EDA)

---

# Analysis & Features

* Medal distribution by country

1. Download the dataset files
2. Open `olympic_analysis.ipynb` in Jupyter Notebook
3. Install required libraries:

   * pandas
   * numpy
   * matplotlib
   * seaborn
4. Run all cells to view analysis and visualizations
